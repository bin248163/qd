# wordpress企业站主题
一个wordpress企业站主题，CSS部分使用SASS编写，完成了PC页面及响应式，浏览器兼容性未做，后台部分还在完善中~
* 可通过函数$clolr定义网站页面主色
* 通过函数$width定义网站pc端显示页面的宽度等，
* 使用部分Html5及CSS3新属性
* 页面中小图标部分全部使用了字体图标，便于修改
* 添加了简单的页面滚动动画
